# Nefer (Dendro/Catalyst/★5) Character Data Implementation

## Overview

v6.1 (Luna II) の新キャラクター Nefer のゲームデータを genshin-calc に実装する。

## Scope

- Nefer の CharacterData 定義（dendro.rs）
- ALL_CHARACTERS への登録
- 天賦バフ（TalentBuffDef）追加（C2 EM自己バフ、C4 敵Dendro耐性デバフ）
- 月兆タレント強化追加（ベネディクションはスタブ実装済み・検証のみ）
- TOML テストケース作成

## Data Source

Honey Impact（gensh.honeyhunterworld.com）— 2026-03-30 取得。

## Design

### 1. CharacterData 定義

`crates/data/src/characters/dendro.rs` に追加。Lauma（同 Dendro/Catalyst/★5）パターン準拠。

- `id: "nefer"`, `name: "Nefer"`
- `element: Dendro`, `weapon_type: Catalyst`, `rarity: Star5`, `region: NodKrai`
- `base_hp: [989.0, 2565.0, 11208.0, 12704.0]`
- `base_atk: [26.81, 69.55, 303.87, 344.42]`
- `base_def: [62.22, 161.41, 705.2, 799.3]`
- `ascension_stat: AscensionStat::CritDmg(0.384)`
- `constellation_pattern: ConstellationPattern::C3SkillC5Burst`

天賦スケーリング:
- 通常/重撃/落下: 全て `damage_element: Some(Element::Dendro)`（Catalyst 共通）
- スキル/爆発: `damage_element: Some(Element::Dendro)`
- **デュアルスケーリング**: スキル・爆発のダメージエントリは ATK + EM の2重スケーリング。Lauma パターン準拠で `_ATK` と `_EM` の2エントリに分割
- Shades の攻撃は純 EM スケーリング（`ScalingStat::Em` 単独）

### 2. 登録

`crates/data/src/characters/mod.rs` の `ALL_CHARACTERS` Dendro セクションに `&dendro::NEFER` をアルファベット順で追加（NAHIDA の後、TIGHNARI の前）。

### 3. 天賦バフ

`crates/data/src/talent_buffs.rs` に `TalentBuffDef` を追加:

| 星座/パッシブ | 効果 | 実装 | BuffType |
|-------------|------|------|----------|
| A1 | 月兆ベネディクション（LunarBloom） | moonsign_chars.rs で実装済み | — |
| A4 | Verdant Dew 強化（EM 500超で+10%/100EM、最大50%） | スキップ（反応DMG%であり StatBuff ではない） | — |
| C1 | Lunar-Bloom基礎DMG +EM×60% | スキップ（反応固有DMG加算、StatBuff外） | — |
| C2 | EM+200（5 Veilスタック時） | **実装** | Toggle → EM +200 self |
| C4 | Shadow Dance中、敵Dendro耐性-20% | **実装** | Toggle → DendroResDown 0.20 |
| C6 | Lunar-Bloom DMG+15% + 追加AoE | スキップ（反応DMG%加算 + proc damage） | — |

### 4. 月兆データ

#### ベネディクション（検証のみ）
`moonsign_chars.rs` のスタブ実装:
- `character_id: "nefer"`, `enabled_reactions: &[Reaction::LunarBloom]`
- `scaling_stat: Some(ScalingStat::Em)`, `rate: 0.000175`, `max_bonus: 0.14`

A1パッシブ「0.0175% per EM (max 14%)」と一致 → **変更不要**。

#### タレント強化（新規追加）
`NEFER_TALENT_ENHANCEMENTS` を新規定義し、`find_moonsign_talent_enhancements` の match に `"nefer"` を追加。

具体的な効果は API から要確認。Lauma の Bloom crit、Flins の Lunar-Charged DMG+20% と同様のパターンで、Nefer は Lunar-Bloom 関連の強化が予想される。

### 5. TOML テスト

`crates/core/tests/data/characters/nefer.toml` を作成:
- 最低 1 ケース: 通常攻撃ダメージ（Dendro、ATK スケーリング）
- 可能であればスキルのデュアルスケーリングケースも追加（ATK+EM 合算が正しく計算されることの検証）
- 手計算で検証可能な値（Lv90、既知ステータス）
