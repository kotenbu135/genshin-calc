# Gambler / 博徒

## Basic Info

- **ID**: gambler
- **Rarity**: 3

## Set Bonuses

**2-Piece**: Increases Elemental Skill DMG by 20%.
**JA**: 元素スキルのダメージ+20%。

**4-Piece**: Defeating an opponent has a 100% chance to remove Elemental Skill CD. Can only occur once every 15s.
**JA**: 敵を倒した時、100%の確率で元素スキルのクールタイムをリセットする。15秒に1回のみ発動可能。
